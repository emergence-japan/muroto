# 室戸ポータル — MUROTO CIVIC PORTAL

高知県室戸市の「政治(市政・議会)」と「経済(人口・産業・予算・財政)」を、
公表された一次資料ベースで一望する**非公式**の静的ポータルサイト。

## 構成

| ファイル | 役割 |
|---------|------|
| `index.html` | 全セクションの本文・テーブル(文章と表のデータはここに直接記載) |
| `style.css` | デザインシステム(深海の藍 × 生成り × 朱のエディトリアル調) |
| `data.js` | チャート用データ(出典付き)。人口推移・年齢構成・産業構成・歳入歳出・ふるさと納税 |
| `charts.js` | チャート描画(SVG折れ線/カラム、div積み上げ・水平バー、ツールチップ、数値表) |
| `main.js` | スクロール表示アニメーションとナビのアクティブ表示 |

ビルド不要。`index.html` をブラウザで直接開くか、ローカルサーバーで:

```bash
python -m http.server 8000
# → http://localhost:8000
```

## データ時点と主な出典(2026年7月3日編集)

- **人口・年齢構成**: 高知県推計人口(令和8年6月1日現在) https://www.pref.kochi.lg.jp/doc/t-suikei/
- **産業構成**: 2020年国勢調査
- **予算(令和8年度)**: 広報むろと2026年6月号「室戸市の財政状況をお知らせします」(p.18-20)
- **財政指標(令和6年度決算)**: 令和6年度財政状況資料集 https://www.city.muroto.kochi.jp/pages/page0071.php
- **市政・議会**: 室戸市公式サイト(選挙開票結果・議会の組織ほか)
- **政策**: 総合振興計画後期基本計画(page3164.php)・各課別ページ・田舎暮らし情報館ほか
- **議会の議論**: 定例会ページ(page0951.php)の付議事件/一般質問/議決結果PDF・議会だより第120号
- **ニュース**: 高知新聞・室戸市観光協会・室戸ジオパークほか(各項目にリンク)

## 更新のしかた

1. **チャートの数値** → `data.js` を編集(人口は毎月、予算は毎年6月頃の広報むろと財政特集で更新)
2. **本文・表の数値**(KPI、市長・議員、財政指標、ニュース、主な事業) → `index.html` の該当セクションを編集
3. 数値を更新したら、フッターと `index.html` ヒーローの「データ時点」も更新すること
4. 「数字で見る市政・議会」の統計ストリップ(議題数など)は定例会ごとに、「分析 — 課題と希望の光」セクションは根拠データを更新した際に記述の整合を確認すること(分析は編集部の見立てである旨を明記済み)

## 編集ポリシー

- すべての数値に出典と時点を付記する
- 未確認情報・裏取りできない情報(Wikipediaのみが出典等)は掲載しない
- 個人の健康情報・進退の憶測は扱わない
- 非公式サイトであることをフッターに明記する
